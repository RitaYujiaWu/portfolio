CoT, ToT, and LLM Reasoner on ZebraLogic

View Project

  • Explored reasoning frameworks such as Chain-of-Thought (CoT), Tree-of-Thought (ToT), and the LLM Reasoners library to solve Zebra Logic Puzzles.

  • Visualized ToT-based solutions, highlighting cases where advanced methods outperformed naive approaches.
